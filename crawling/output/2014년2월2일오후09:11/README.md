# 2014년 2월 2일 오후 09:11
사십년쯤 전 이려나ᆢ할매 하고 아버지  밥묵는데  달걀 먹고 싶어 밥안무다가  엄마한테 두들겨 마꼬  중하교  일학년때  할머니 돌아 가시고 살밥 먹고 달걀먹고ᆢ뽀얀  살밥에  국물도 있고 ᆢ 넘 몰랐다 ᆢ그날이 할매 가는줄을 ᆢ초등하꾜  다닐때는 소죽도  끌려야하고  타잔도 보고싶고 서부소년 차돌이도 보고싶고ᆢ텔 레비는 없고 ᆢ소꼴 비러  마이도 다녔다  소꼴 비가꼬 갔다주마 텔레비  살짝 보여주고ᆢ 
육영수 여사님 떠나실땐  텔레비 실컨보고 한없이 울었다 ᆢ
그해 구월에 아버지 사과 팔았는데 돈은 갖고오지 않고 그냥오셨다  텔레비 하나들고  엄마는 화가나도  참고  계시고 난 좋아서 안텨나 설치하고ᆢ 지금와서  생각지만  서부소넌  차돌이ᆢ마린보이ᆢ보고싶어서 백곡 뒷동네  안테나 들고 다닌게 엇그제구나 ᆢ 이젠 타잔도ᆢ마린보이도ᆢ
화롯불 들고가면 할배가 주신 사탕이 그립네ᆢ
